## Core Highlights

### 🎯 AI-generated TOEIC passages in real time

Based on your target score (500-900) and chosen topic, AI instantly creates a TOEIC-style passage with:

- **Sentence-by-sentence EN/ZH translation**
- **Key vocabulary cards** (part of speech, IPA, Chinese meaning, examples)
- **Useful phrase explanations**

### 🔊 AI voice playback + sentence-level highlighting

- Natural speech generated by Google Gemini TTS or OpenAI TTS
- **Multiple selectable voice styles** (Gemini and OpenAI each provide 6 voices: clear female, soft female, lively male, calm male, etc.)
- **Sentence sync highlight** during playback for listening + reading training
- **0.25x / 0.5x / 0.75x / 1.0x playback speed**
- Replay any sentence independently

### 🗣️ AI speaking conversation (new)

- Start speaking practice in one tap and talk with AI in English
- Multiple TOEIC scenarios built in (airport, meeting, customer service, interview, etc.)
- Conversation history is saved automatically for review

### 📝 Mock exam (new)

- Generate complete mock tests by target score (listening / reading / vocabulary / grammar)
- Submit directly for instant results and error review
- Supports AI-generated explanations for wrong answers

### 🧠 SRS spaced repetition memory system

This is the key feature that helps you truly remember words:

- After saving words, review is auto-scheduled at **0 -> 1 -> 3 -> 7 -> 14 -> 30 days**
- Three rotating question types: **EN->ZH / ZH->EN / listening multiple choice**
- Correct answers level up, wrong answers level down, focusing on weak words
- Lv.0 to Lv.5 labels for clear progress tracking

### 📖 Long-press dictionary lookup

Long-press **any word** in a passage to see part of speech, IPA, Chinese meaning, and example usage. If a word is missing, tap once for **AI instant analysis**.

### 🔀 Dual AI providers, switch freely

- Choose **Google Gemini** or **OpenAI** as your API provider in settings
- Built-in list of common models (e.g. Gemini 2.5 Flash / Pro, GPT-4o mini / GPT-4o), with support for **manually entering a model name**
- Text generation, voice playback, mock exams, and speaking conversation all use the selected provider
- A **Test API Key** button in settings lets you verify your key works right after pasting it

### ☁️ Google cloud backup

Sign in with Google to back up your learning records and vocabulary book to Google Drive. Supports **Back Up Now / Restore from Cloud / Sign Out** (API key excluded).

## How can I use it for free?

This project supports both **Google Gemini** and **OpenAI** as AI providers. Both have free usage options — just get an API key and paste it into settings to start learning.

### Option 1: Google Gemini (completely free)

The [Google Gemini API](https://aistudio.google.com/app/apikey) offers a **free daily quota** that is more than enough for personal learning.

> Your API key is stored only on your local device (IndexedDB) and is not backed up to Google Drive.

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey) and get a free API key with your Google account
2. In the app settings, select **Gemini** as your provider and paste the key
3. Start learning!

### Option 2: OpenAI (free with data sharing)

The [OpenAI API](https://platform.openai.com/api-keys) is usage-billed, but if you allow OpenAI to use your inputs and outputs for training and improvement, you receive **free complimentary tokens every day** (see the [OpenAI help page](https://help.openai.com/en/articles/10306912-sharing-feedback-evaluation-and-fine-tuning-data-and-api-inputs-and-outputs-with-openai) for details).

1. Go to [OpenAI Platform](https://platform.openai.com/api-keys) and get an API key (requires a minimum top-up)
2. In your account settings, enable "Share inputs and outputs with OpenAI" to unlock daily free tokens
3. In the app settings, select **OpenAI** as your provider and paste the key
4. Start learning!

## User Guide

### Step 1 - Set API provider and API key

On first launch, a setup dialog appears automatically:

- First choose your **API provider** (Google Gemini or OpenAI) and **model** (pick from the list or enter a model name manually)
- Click the link to get an API key from [Google AI Studio](https://aistudio.google.com/app/apikey) or [OpenAI Platform](https://platform.openai.com/api-keys), paste it, and save
- Use **Test API Key** to verify the key works; to reset, use **Clear in one click** next to the input

### Step 2 - Choose a mode in Practice

Switch between **Reading Practice / Speaking Conversation / Mock Exam** based on your daily goal:

- **Reading Practice**: choose target score + topic, then click **Start Learning**
- **Speaking Conversation**: choose a scenario or custom topic, then click **Start Speaking Conversation**
- **Mock Exam**: choose target score, then click **Start Mock Exam**

### Step 3 - Reading mode: read + listen

After AI generates a passage, you are automatically taken to the learning page:

- Tap play to listen with sentence highlighting
- Toggle **Show Chinese** for side-by-side translation
- Toggle **Hide English** for listening-only training
- Long-press any word to view details

### Step 4 - Speaking / Mock exam: fix weaknesses fast

- **Speaking conversation**: real-time back-and-forth with AI, auto-saved in records
- **Mock exam**: submit answers for instant score, plus one-click error explanation

### Step 5 - Save words and start SRS review

See an unfamiliar word while reading? Tap the bookmark icon to save it to your vocabulary book. Once you have at least 3 words, start SRS review in the **Vocabulary** tab with three rotating question types until words stick.

### Step 6 - Optional cloud backup

After signing in on the settings page, you can use **Back Up Now / Restore from Cloud / Sign Out**. Keep your records and vocabulary when switching devices.
